# Moving Window

**Moving Window** is a silly little Minecraft mod that makes your game window move around when your player moves.

Walk forward? The window scoots. Jump? The window joins in. Fall off a cliff? Your desktop may briefly become part of the adventure.

It is not trying to be useful. It is trying to be funny. And honestly, it is doing its best.

## What does it do?

Moving Window turns ordinary player movement into actual window movement on your screen. As you wander, sprint, hop, stumble, and otherwise make questionable decisions in Minecraft, the game window reacts by sliding around your desktop.

It is basically motion sickness for your operating system.

## Features

- Moves the Minecraft window as your player moves in-game.
- Supports horizontal movement and vertical movement.
- Includes configurable movement multipliers.
- Automatically stays out of the way while the game is fullscreen.
- Client-side only, so servers do not need to install it.
- Excellent for chaos, comedy, and confusing anyone watching over your shoulder.

## Requirements

- **Minecraft:** 1.20.1
- **Mod loader:** Forge
- **Environment:** Client side

This mod is intended for the regular desktop version of Minecraft. Since it physically moves the game window, it is not very meaningful in fullscreen mode.

## Installation

1. Install Minecraft Forge for Minecraft 1.20.1.
2. Download the Moving Window `.jar` file.
3. Place the `.jar` file in your Minecraft `mods` folder.
4. Launch the game.
5. Move around and watch your window develop a mind of its own.

If nothing seems to happen, make sure Minecraft is running in windowed mode rather than fullscreen.

## Configuration

Moving Window includes client-side configuration options so you can decide how dramatic the nonsense should be.

You can adjust:

- **Horizontal movement multiplier**  
  Controls how strongly left/right-style movement affects the window.

- **Vertical movement multiplier**  
  Controls how strongly jumping, falling, and other vertical movement affects the window.

Set the multipliers lower for a gentler wobble. Set them higher if you believe your desktop icons have grown too comfortable.

## Recommended usage

For the best experience:

- Play in windowed mode.
- Start with the default settings before making things more extreme.
- Do not place the window somewhere important and then sprint directly into danger.
- Maybe warn your friends before showing them. Or do not. That is also funny.

## Multiplayer

Moving Window is a **client-side mod**. You can use it while joining servers, and the server does not need to have it installed.

The mod only changes how your own game window behaves on your own computer. Other players will not see your window scooting around unless they are physically nearby, in which case they may have questions.

## Fullscreen note

Moving Window does not move the game window while Minecraft is fullscreen. This is intentional: fullscreen windows generally do not enjoy being dragged around like a shopping cart with one bad wheel.

Switch to windowed mode if you want the full chaotic experience.

## Safety and sanity notes

This mod is designed as a joke/novelty mod. It may be distracting, surprising, or deeply unhelpful during serious gameplay.

Use responsibly, especially if:

- You are recording or streaming.
- You are prone to motion sickness.
- You are in the middle of a delicate parkour section.
- You value calm, order, or windows staying where you left them.

## FAQ

### Is this required on servers?

No. Moving Window is client-side only.

### Does it work in fullscreen?

No. Use windowed mode if you want the window to move.

### Can I reduce the movement?

Yes. Lower the movement multipliers in the client config.

### Can I make it more ridiculous?

Also yes. Increase the multipliers. This is not legal advice.

### Why would anyone make this?

Because sometimes Minecraft needs fewer sensible features and more desktop-based slapstick.
